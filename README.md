```
pkg upgrade -y && pkg update -y && pkg install python -y && pkg install nodejs-lts -y && pkg install nodejs -y && pkg install git -y && pkg install ffmpeg -y && pkg install wget -y
```

**2° Comando**
```

```

```
cd /sdcard/

```

```
cd /sdcard/sakura-botv6
```
